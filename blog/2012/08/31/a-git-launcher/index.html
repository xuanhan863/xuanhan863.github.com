A Git Launcher

a git launcher : )

当你输错后, 有一个ASCII码的terminal动画效果. 之后, 它会把你的意思正确执行.

只有一个C 文件的实现, 只针对git命令.

$ git clone https://github.com/rwos/gti.git
$ make
$ make install # as root, probably

参考: http://r-wos.org/hacks/gti
